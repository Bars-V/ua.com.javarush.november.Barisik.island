Программа начитается с класса Fight
класс Fight обращается к классу Initialization и через него стартует все основные загрузки (пока в разработке)

Все параметры всех классов хранятся в классе Parameters
первоначально их туда загружает класс FillTable (метод startTable) (пока временно все параметры в виде хардкода)

функции всех животных такие как охотится есть голодать выбирать направление
и их поля прописаны в абстрактном классе Animal
фактически все классы относящиеся к животным Alive, Predator, Herbivore и классы животных пустые, кроме класса Animal
класс Alive нужен для корректного извлечения животных из ячейки острова

класс CarnivorousBush в разработке

класс Constants хранит в себе текстовые переменные

Класс Land представляет собой хранилище ячеек на 10_000_000 ячеек по горизонтали и вертикали.
Имеет методы записи, удаления и вывода данных по ячейкам и по острову в целом
